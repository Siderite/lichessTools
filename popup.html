This extension turbocharges LiChess with a lot of useful and fun features.

You can access the options for the extension in the LiChess preferences. Many of these are not enabled by default, so it's important for you to go there. A good resource is the LiChess Tools user manual page.

Find the latest updates and keep up to speed by joining team LiChess Tools Users and/or reading the team forum.
